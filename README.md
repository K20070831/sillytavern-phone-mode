# SillyTavern 手机模式扩展

在 SillyTavern 中召唤一个持久化的手机聊天界面。

## 安装方法

1. 打开 SillyTavern，点击顶部「扩展」图标
2. 找到「从 URL 安装」输入框
3. 粘贴以下链接并安装：

https://github.com/K20070831/sillytavern-phone-mode

## 使用方法

- 输入 `/phone` 召唤手机界面
- 用 `/` 分隔多条消息，例如：`你好/在吗`
- 转账：`(转账+50)`
- 图片：`(图片+一只猫)`
- 点「结束通话」退出手机模式
